# Doom AI Bot (Reinforcement Learning)

Train modern AI behavior in the classic Doom engine using reinforcement learning.

This project demonstrates how reinforcement learning can evolve enemy behavior in classic FPS games.

Instead of scripted enemies, the AI learns movement and combat behaviors using PPO.

---

## Features

• VizDoom environment  
• PPO reinforcement learning  
• Parallel training environments  
• TensorBoard monitoring  
• Checkpoint saving  
• Trained combat bot  

---

## Installation

Clone the repo:

git clone https://github.com/YOURNAME/doom-ai-bot

Install dependencies:

pip install -r requirements.txt

---

## Train the AI

python train.py

---

## Watch the AI play

python watch_ai.py
